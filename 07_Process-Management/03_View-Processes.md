## Viewing Process Details
---

### Using `ps`

Show processes for a specific user:

```bash
ps -u username
```
---

Show a process by name:

```bash
ps -C processname
```
---

### Using `pgrep`

Find a process by name and return its PID:

```bash
pgrep processname
```

---

### Using `pidof`

Find the PID of a running program:

```bash
pidof processname
```
